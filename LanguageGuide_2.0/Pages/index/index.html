Welcome

This website aims to help those with knowledge of Scratch to learn new programming languages.

The languages we are currently doing tutorials on are C# and JavaScript.

So let's get started!


C# is a programming language developed by Microsoft.

If you want to develop desktop applications or console games in C#, you should start with Unity.




JavaScript is the programming language of HTML and the web. It is used to program the behavior of web pages.

If you're into web application development or developing web based games in JavaScript, you should start with Phaser.




An IDE (Intergrated Development Environment) is a software application that allows developers to write and test their software easily.

The IDE we will be using is Visual Studio Code.
