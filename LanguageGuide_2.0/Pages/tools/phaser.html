What is Phaser?

Phaser is an open source HTML5 game framework by Photon Storm. It is designed to create games that will run on desktop and mobile web browsers.


To be able to access and use the Phaser library in your code, you must do the following everytime you create a Phaser project:

1. Create a folder for your project.

2. Inside the folder create another folder called js.

3. Download phaser.js file from phaser.io.


4. Cut and paste the file into the js folder.

Make sure you have installed Visual Studio Code.
