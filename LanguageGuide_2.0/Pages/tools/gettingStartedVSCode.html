What is Visual Studio Code?

Visual Studio Code is an IDE by Microsoft that allows you to write and test your code. It supports many programming languages so it is a good IDE to get started with.

First download and install Visual Studio Code and .NET Core.



Once you have installed Visual Studio Code and .NET Core, the first thing you need to do is install C# extension to allow you to write and test your code in C#

Open Visual Studio Code, Go to View > Extensions, then type in C# and select C# for Visual Studio Code(powered by OmniSharp) to install.




Go to File > Preferences > Color Theme




The default language mode in Visual Studio Code is Plain Text

To change it, go to the buttom right-hand corner of the editor and select Plain Text box



A text box will appear at the top center of the screen, type in the language you want and select
